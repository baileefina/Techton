Hyperlinks



Hyperlinks (a tags) They allow you to jump from one website to another, whether it's to another part of the same website, a different website entirely, or even a specific file like a document or image.

Hyperlinks are created using < a > tags. the href attribute with the tag specifies the destination of the link

Note: We will be using the target attribute "_blank" or "_self" in order for this to work. The target attribute makes it so that it will open a new tab for the link
