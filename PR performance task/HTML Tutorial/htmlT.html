HTML basics



DOCTYPE Declaration tells the web brower what version of HTML you're using. For example, !DOCTYPE html tells the browser to intepret the Document as HTML.

HTML Tag Is the main container for your entire webpage. Everything inside the html tags is part of your webpage.

Head section It contains information about the page that isn't directly displayed on the screen, but is important for the browser and search engines to understand. usually included title tags, meta tags, link tags, script tags

Body section This is the actual content of your webpage. It contains all the visible elements that users see, such as: headings, paragraps, images, links, lists, tabls, forms tags
