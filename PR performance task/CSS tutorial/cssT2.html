Fonts



Fonts are the building blocks of text on the web. they determine how letters, numbers, and symbols look on your website. Just like colors.

Here are different Font Attributes:

  1. Size Specifies the font size of text.
  2. Style: Specifies the font style for text.
  3. Spacing: The amount of space between letters in a font.
  4. Weight: The thickness of boldness of a typeface.
