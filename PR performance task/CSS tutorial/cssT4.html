Margins



Margins Are the space around an element. They control the distance between an element and its neighbor. Margins are transparent, so they don't effect the background color of the element.

You can set margins for all sides of an element using margin properties:

  1. margin spaces out from all sides
  2. margin-top spaces out from the top
  3. margin-bottom spaces out from the bottom
  4. margin-left spaces out from the left
  5. margin-right spaces out from the right
