Basic CSS coloring



CSS (Cascading Style Sheets). is the language we use to style our web pages. One of the most basic things you can do with CSS is to change the color of elements on your page. Which is what we are gonna be tackling.

There are three ways to styling HTML:

  1. Inline: Using the style="." attribute inside the opening tag
  2. Internal: Using the "style" tags inside the head tags
  3. External: Using a separate file with the .css, and linking is using link
