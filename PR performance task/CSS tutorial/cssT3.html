Borders



Borders are frames around your website elements, giving them definition and visual separation. Think of them as the lines that outline a picture.

Here are different Border-Styles Attributes:

  1. None/Hidden: No Border.
  2. Dotted: Dotted line border.
  3. Solid: Dashed line border.
  4. Double: Double line border.
  5. Groove: Groove style border.
  6. Ridge: Ridge style border.
  7. Inset: Inset style border.
  8. Outset: Outset style border.
