Variables



What are Variables? Imagine variables as containers that hold information. They are essential building blocks in any programming language, think of it this way: you have a box, and inside that box, you can store different things, that's what variables are. You can put numbers, words(strings) or even complex data structure inside these variables.

Here's how it works

  1. Declaration: You start by declaring a variable using the let, const, or var keywords. For example: let age; or const name = "Alice";.
  2. Assignment: You assign a value to a variable using the assignment operator (=). For example: age = 25; or name = "Bob";.
  3. Data Types: Variables can hold different types of data, such as numbers, strings (text), booleans (true/false), arrays (lists), and objects (collections of data).
  4. Reassignment: Variables declared with let can be reassigned a new value. For example: age = 25; and later age = 30;. Variables declared with const cannot be reassigned.

By using console.log(), you can:

  • Store and retrieve information
  • Perform calculations

  • Make decisions
